# Power BI Sales Data Analysis Dashboard

## Project Overview

This project presents an interactive **Sales Data Analysis Dashboard** developed using **Microsoft Power BI**. The dashboard analyzes sales performance across regions, states, products, and retailers to identify business trends and key performance indicators.

The goal of this project is to transform raw sales data into meaningful insights using data visualization and business intelligence techniques.

This dashboard helps users quickly understand **sales performance, operating profit, units sold, and regional sales distribution**.

---

# Project Objectives

The main objectives of this project are:

* Analyze **total sales performance**
* Evaluate **operating profit and operating margin**
* Identify **top-performing products**
* Compare **sales across different regions**
* Track **monthly sales trends**
* Analyze **retailer sales performance**

---

# Tools and Technologies Used

* **Microsoft Power BI** – Data visualization and dashboard creation
* **Microsoft Excel / CSV Dataset** – Data source
* **Data Modeling & DAX** – For measures and calculations

---

# Dataset Description

The dataset used in this project contains **Adidas sales data** across different regions and retailers.

### Key Fields in the Dataset

* Invoice Date
* Region
* State
* Product
* Retailer
* Units Sold
* Price per Unit
* Total Sales
* Operating Profit
* Operating Margin

These fields allow analysis of sales performance across multiple business dimensions.
